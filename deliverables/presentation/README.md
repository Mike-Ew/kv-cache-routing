# Presentation

This folder contains the slide-deck source and supporting materials for the project presentation.

## Files

- `presentation/deck.md`: slide deck in Markdown (Marp-compatible).
- `presentation/timeline.md`: 2-week execution plan up to the presentation due date.
- `presentation/speaker_notes.md`: notes to help present the deck.
- `presentation/assets/`: any images made specifically for slides.
- `presentation/references/`: drop reference papers here (PDFs are gitignored).

## Editing Workflow (Recommended)

1. Edit `presentation/deck.md` first.
2. As you add figures/results, link assets from `presentation/assets/` or reuse `paper/figures/`.
3. Keep slide titles stable so speaker notes stay aligned.

## Rendering Options

- If you use Marp (VS Code extension or CLI), `presentation/deck.md` already uses slide separators (`---`).
- If you do not use Marp, treat `presentation/deck.md` as an outline and paste slide-by-slide into Google Slides/Keynote.
